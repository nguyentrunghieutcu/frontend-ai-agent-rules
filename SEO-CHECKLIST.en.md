## Google SEO

Google can index GitHub README pages, but a public landing page is better.

Recommended title:

```txt
Frontend AI Agent Rules Kit for Claude Code, Codex, Cursor & Copilot
```

Recommended meta description:

```txt
Practical frontend AI agent rules for Claude Code, Codex, Cursor, Gemini, Windsurf and GitHub Copilot. Reduce noisy PRs, prevent broad refactors and keep AI code aligned with React, Next.js and TypeScript project patterns.
```

Recommended landing page slug:

```txt
/frontend-ai-agent-rules
```

Recommended H1:

```txt
Frontend AI Agent Rules for Claude Code, Codex, Cursor and Copilot
```

Recommended sections:

- Problem: AI agents create noisy diffs without rules.
- What's included: `CLAUDE.md`, `RULE.md`, templates and checklists.
- Supported tools: Claude Code, Codex, Cursor, Gemini, Windsurf, Continue, Copilot.
- Supported stacks: React, Next.js, Vite, TypeScript, frontend monorepos.
- Free version CTA.
- Pro version CTA.
- Team Setup CTA.
